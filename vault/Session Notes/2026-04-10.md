## Key Decisions

### Generic Vendor Branding (FIRM)
- No vendor names anywhere in the rental UI — not JOY, not anyone
- Branding is always "Golf Cart Rentals" / "Port A Local"
- Reason: protects the model — vendor can swap without customer noticing, not locked to any single vendor

### Portal Strategy (vs. Directory Listings)
- Beach and maintenance revenue should stay in-house via portals, not routed to third-party directory listings
- Same model as golf carts: customer submits through us, we handle the booking, we collect the fee

### Nav Cleanup — Decisions Made
