## Architecture position

PAL is now configured as a **Tier 1 design-contributor-friendly tenant** under the agent-driven autonomy principle (`feedback_agent_driven_contributor_autonomy.md`). Specifically:

- **Cosmetic / brand / copy / component edits** → push branch → PR → Vercel preview builds → checks pass → **auto-merges** (no human review needed)
- **Backend / API / auth / DB / payment / build-config edits** → CODEOWNERS auto-requests `@haveebot` + `@nickbmerrill-collab` review → operator approves → merges

Same architecture will apply to every HeyeDeploy customer's design contributor when v1.0 ships. PAL is the canonical proof.
