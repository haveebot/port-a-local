Operator (Winston) gets visibility (cron + admin@ email); Nick + Collie don't see each other's stuff. CONTRAST: super-admin revenue pings DO go to all three because PAL revenue is shared visibility.
- **Agent name is Havee.** Winston + Nick high-school nickname → cross-Heye-Lab agent name. Sign as `- Havee` for insiders. Customer-facing SMS still entity-only ("— The Port A Local"). Memory: `feedback_havee_naming.md`.
- **Twilio Lookup is a starting filter, not the final word.** Worldcall (Tarpon's 988-8161) and Charter Fiberlink (Jackfish 339-1089) classify as mobile / fixedVoip per Lookup but bounce 30006 in production. Always verify with a real send before relying on a number.
- **Vercel function env reads must be per-invocation, not module-load.** Shipped a fix mid-session: `runInsiderAgent` now reads `process.env.ANTHROPIC_API_KEY` inside the call, not at module top. Module-level captures the value at first cold-start and warm instances retain stale empty even after redeploys. **Apply this pattern for any future env that's added post-build.**
- **Vercel's serverless background tasks die on response return** for multi-second work. Single-HTTP-call fire-and-forget (Resend email forward) survives. Multi-step Claude agent loop does not. Solution: AWAIT inside the webhook handler. Twilio's 15s budget covers it.
- **Anthropic API key in Vercel via `--value` flag, not stdin.** Same trailing-newline bug as the WHEELHOUSE_TOKEN rotation per memory. Use `vercel env add ANTHROPIC_API_KEY production --value="<key>" --sensitive --force`.
- **CRON_SECRET fix unblocked the daily Pulse cron** (silently broken for weeks). PAL Pulse digest will fire 8am CT tomorrow for the first time in production.
- **Lead-blast SMS format = chunked line-spaced blocks** per Collie's revisions. Each line carries one fact. Same format applied to beach-vendor blast.
