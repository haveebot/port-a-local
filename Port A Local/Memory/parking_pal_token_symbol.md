**Topic:** Develop a standardized typographic token symbol for Port A Local — functioning the way ₿ represents Bitcoin, $ represents dollars, or Ⓜ represents McDonald's in casual reference. Not a cryptocurrency symbol; a character-level brand mark.

**Why it matters:** If PAL becomes a recognized media/platform brand, having a single-glyph shorthand is a compounding brand asset. It shows up in URLs, in nav, in inline text, in merch, in typographic uses where the full lighthouse mark is too much. Every mature brand at scale has one — the question is whether we pick it early and seed it, or hope one emerges.

**Starting questions when we pick this up:**
1. What glyph? Lighthouse silhouette at 1em? A stylized "P" or "A"? A custom ligature like Æ? A Unicode symbol we can squat on? A novel Unicode proposal (long-tail)?
2. How do we use it early so it becomes recognized? Coordinates block, footer, email signatures, comments.
3. Does it need to be typable on a standard keyboard, or is copy-paste / font-embedded acceptable?
4. Does it play with the existing lighthouse mark, or replace it in specific contexts?
